# FP division abstraction design

The design note uses the Springer LNCS format required by CAV. The official
Springer template files `llncs.cls` and `splncs04.bst` are included locally so
the document is self-contained.

Build it from this directory with:

```bash
latexmk -pdf fpdiv_abstraction_plan.tex
```

The two requested chapters are represented as Sections 1 and 2 because LNCS
conference papers use `\section` as their top-level division.
